解释上述代码

1.window表示windows对象，来源于默认的命名空间，grid同理，xaml语言每看见一个标签就创建一个对象。

2.xmlns=”http://schemas.microsoft.com/winfx/2006/xaml/presentation"表示引入一个命名空间，没有起名的原因是可以简化写法，因为window，grid等这些对象均来源于它，如果起名字，则每个对象前面需要使用“y:”的形式，比较麻烦。但是一个xaml文件中仅能有一个默认名称的写法

3.xmlns:x=”http://schemas.microsoft.com/winfx/2006/xaml" 引入xaml命名控件，并命名为x

4.x:Class=”WpfHelloworld.MainWindow表示xaml生成的代码要与后台的c#代码进行合并。

xmal语言写的代码代码会生成一个类，后台c#代码也会有一个类，两者代码会进行合并，所以后台的c#使用partial修饰，表示两者生成之后会自动合并到一个类里边，就是mainwindow类，若前后台名称不一样，则会生成两个类，两者之间没有关系。

x:classModifier;访问级别，前台的访问级别必须与后台的访问级别相同。

x:name;为控件起一个名字。这样后台代码可以之间使用控件的名字进行控件操作。如果是wpf中的自有的控件，则使用x:name和使用name是完全一样的，如果是是后台的实体类被引入xaml中，则必须使用x:name的形式。

5.MVVM：就是将mvc中的C变成了VM，他与View是一一对应的关系。这个一般用于大型企业项目中。对于比较小的程序可以不使用MVVM。

不使用mvvm，则如下格式：事件直接写在控件里面。优点是简单，缺点就是如果前端改变，则后端代码需要改变，维护比较麻烦。

title:blazor

Blazor Server 托管模型具有以下优点：

  • 下载项大小明显小于 Blazor WebAssembly 应用，且应用加载速度快得多。
  • 支持瘦客户端。 例如，Blazor Server 应用适用于不支持 WebAssembly 的浏览器以及资源受限的设备。
  • 应用的 .NET/C# 代码库（其中包括应用的组件代码）不适用于客户端。

Blazor Server 托管模型具有以下局限性：

  • 通常延迟较高。 每次用户交互都涉及到网络跃点。
  • 不支持脱机工作。 如果客户端连接失败，应用会停止工作。

Blazor WebAssembly 托管模型具有以下优点：

  • 没有 .NET 服务器端依赖，应用下载到客户端后即可正常运行。
  • 工作可从服务器转移到客户端。

Blazor WebAssembly 托管模型具有以下局限性：

  • 应用仅可使用浏览器功能。
  • 需要可用的客户端硬件和软件（例如 WebAssembly 支持）。
  • 下载项大小较大，应用加载耗时较长。

title:matlab

命令：

针对某个函数的使用方法，可以搜索，然后点击里面的example

关闭matlab：quit或者exit

matlab大部分函数只要能传入一个变量，那么就可以传入一个向量。例如sin30，sin([30 60 90])会分别计算出sin30 sin60 sin90即批量处理数据

暂停20s：pause(20)

停止运行命令：ctrl+c

恢复窗口布局：主页=》布局=》默认

清屏：clc

删除所有变量：clear

删除执行的变量：clear temp，a，b；删除指定的三个变量

删除某个元素值：将该位置的值赋值为空[ ]即可

列出所有变量及其详细信息：whos

单精度(默认)：single() 双精度：format long 后接4位小数：format short

在脚本中写代码，命令行中调用该脚本（直接输入脚本名,或者直接run（F5）），命令行窗口的作用是用来显示结果，加分号回车后会换行，不加分号回车后会显示结果

根据几个字母弹出所有相关的函数名：使用tab键

3X10^8：3e8

换行符： +…表示换行接着输入

取绝对值：abs()

三角函数sin(pi)，tan(pi),cos(pi),使用help elfun命令来提示所有的数学函数或者点击输入处左侧的fx。默认为弧度，若使用角度为sind(180)。sin（[1 2 3]）会分别计算处sin1,sin2,sin3

读取脚本：load newfile.dat

常用函数

1.向0取整：fix(-3.5) (-3) fix(1.8) (1)

2.向负无穷取整：floor(-3.5) (-4)

3.向负无穷取整：ceil(16.99)(17)

4.四舍五入：round(1.5)(2) round(pi,2)(3.14保存两位小数)

5.求余：mod(13,3)(1) 或者rem(13,3)(1)

6.符号函数sign()

7.开根号sqrt()

8.求根nthroot(2,3),2的立方根

9.角度转弧度：deg2rad(180) rad2deg(pi)弧度转角度

10.对数：log(2.718)（以e为底） log2(1024)（以2为底）

正无穷：Inf或者1/0 负无穷: -Inf或者-1/0 是个常量

NaN：not a number 常量

rand():0-1之间的随机数，默认随机种子为1，根据种子生成随机数

rand(2);返回一个2x2的矩阵，每个元素都是0-1

rand(2,3);返回一个2行3列的矩阵

randi([1 10])指定从1到10的整数

randi([1 10],3,4);3行4列的矩阵，每个元素都是1-10

rng(1);随机数种子设为1

生成0-100的整数：randi([0.100])
