EF,Linq

Enitity framework (ef)学习

ef就是一个设计器

属于ORM；O：DOmainObject领域模型

​ R；Ralation Database 关系型数据库

​ M：Maping映射关系

ORM就是将相应的实体映射到相应的数据库表，然后使用orm框架封装好的api进行数据库访问，减少了自己写数据库访问类的步骤**。

三种开发方式： 1.dbfirst：先创建数据库再开发（不推荐）

​ 2.modelfirst：先创建模型再创建数据库（不推荐）

​ 3.codefirst:代码优先（推荐）

以下都是根据dbfirst开发方式进行的。

建立数据库，数据表：设置主键：选中id，在右侧的属性的标识中，将false改为true即可

在解决方案中建立一个控制台项目，选中项目创建新建项选择ADO .net实体模型，选择来自数据库实体模型，连接已经创建好的数据表即可创建出可视化模型

如果修改数据库，则不能从该可视化模型中操作（就是自动创建的edmx文件，也称ef设计器），需要从数据库中操作，修改完成之后，在界面右键，从数据库中更新模型可以选择添加 修改，删除对应表中的数据，此时可视化界面中修改，但是model.edmx中代码没有改变，需要点击保存，后才会自动更改。（即要想保证数据库操作与可视化界面中的操作同步，先更改数据库，然后操作右键操作界面中增删改，在保存）

如图，context是上下文类，即可成立DbContext类，里面封装了以连接数据库的方法

context.cs的DbSet里面里面写了很多增删改查的方法，即DbFirstEntitiels简称**上下文类，这再程序中是核心**，在程序中调用的所有的方法都是上下文类中的方法。

然后使用在控制台程序中使用实体执行增删改查操作

Linq

linq to object (数组，list)–查询内存里面的数据，传统方法是使用for或者foreach循环，这里使用linq

linq to sql(查询数据库用的)

自定义linq不常用，官方定义的linq用于查询的是var list = student.Where(x=>x.Age<20).ToList();此表达式表示再Student这个集合/数组/数据库中成员定义为x，其中x的年龄小于20，此时只是一种写法返回的是IEnarable类型，要想获得查询结果需要加入toList（）方法（student定义的是List类型），否则返回的是IEnurable类型，加入之后返回的才是具体的Student类型

数据库

技巧

连接到服务器：指的是存放数据库的那个电脑，数据库服务器名称就是右键我的电脑 名称，一般数据库服务器只有一个计算机名称，所以通常不做更改（同一网段中可以连接其他服务器）

连接到数据库：指的是与某个指定的数据库进行连接的过程，建立成功后在服务器资源管理器中可以看到连接，默认连接的是master数据库

添加sqlserver：指的是创建一个sqlserver管理界面，将与指定的服务器进行连接将所有的数据库导入到sqlserver中在vs中放在sqlserver对象管理器中

连接字符串：以上都是通过可视化界面进行连接数据库的，但是编程必须使用代码进行数据库的连接，所以通过连接字符串连接数据库，连接字符串寻找方法：数据连接中右键属性，在属性对话中的可以找到连接字符串，如下

“Data Source=(localdb)\MSSQLLocalDB;Initial Catalog=master;Integrated Security=True”

连接的服务器为(localdb)\MSSQLLocalDB，连接的数据库为master

1.打开链接：等价于在命令行中输入mysqli -u username -p password进行连接数据库。注意，不是一个数据库对应一个密码，而是一个用户一个密码对应一个连接名，这个连接名可以随便起，只是方便客户进行用户名分类，要不然使用用户名密码的形式查看不方便，一个连接名里面可以创建多个数据库，使用某个数据库不需要密码因为之前已经进行了连接。在同一个连接名下的数据库都是用同一个用户名和密码。一个用户对应一个连接名。

2.在vs中无法为表添加数据的原因是：选错了项目，应该选择sqlServer，而不是项目

3.右键表：查看数据，为表添加数据

虚拟机设置

vmnet实际上表示虚拟交换机，一共vmnet1-vmnet19，将系统连接到某个交换机上，相当于机房中的真实机有一根网线连接到真实交换机上。每个vmnet对应一个虚拟网卡。其中vmnet0表示的是真实机所连接的交换机，如果连接到vmnet0，则相当于该系统从虚拟机跳出来直接连接到真实交换机，可以使用该虚拟机完全当作真实机使用。
1.使用桥接模式：使用的真实机的网卡，可以连接互联网也可以实现虚拟机和同网段其他真实机互联，可以将其看作一个真实机，但网段必须与真实机一样。

技巧

MVC5：新建项目的时候选择web项目-》mvc

更换访问端口：右击项目-》属性-》web

​ Models：实体模型，就是用来操作数据库的

​ View：视图，用户可以看到的前端界面

​ Controllor：决定用户调用哪个视图，调用哪个逻辑计算

vs2013在iis上发布一个网站：

​ 1.首先点击项目于右键-》发布-》自定义-》输入一个发布名-》发布方法选择文件系统-》目标位置选择当前项目所在的路径名-》点击发布完成。发布就是将网站源码打包成一个文件夹，将源码封装起来。不让用户看见源码，例如写完的exe文件就是已经打包好的。
